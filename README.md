# is Valid CPF Number

Validation for CPF Number input.


## Requires

- [Node.JS](http://nodejs.org/)


## Usage

```javascript
var cpfInput = getElementById(#contact-cpf);
isValid.cpf(cpfInput);
```


## Includes

- [Mocha](http://visionmedia.github.io/mocha/)
- [Chai](http://chaijs.com/)
